# HSM.03 Send the reply through your inbox

## HSM.03.1 Open your inbox

- Use the agent-manager MCP tool to access the manager drive (ID: {{documents.driveId}})
- Open your inbox document (ID: {{documents.inbox.id}}) through the agent-manager tool
- Locate the thread with id: {{thread.id}}

## HSM.03.2 Add your reply to the thread

- Use the agent-manager MCP tool to add your reply to the thread
- Paste the reply you formulated in HSM.02
- Ensure the reply appears correctly in the conversation flow
- Check that formatting and any references are preserved
- Save the updated inbox document through the agent-manager tool

## HSM.03.3 Mark the original message as read

- Use the agent-manager MCP tool to mark message {{message.id}} as read
- This prevents reprocessing the same message on future inbox updates
- Confirm the read status has been updated in the inbox document
- Save the updated inbox document through the agent-manager tool

## HSM.03.4 Verify the reply was sent

- Confirm your message appears in the thread
- Check that it's properly linked as a reply to the stakeholder's message
- Ensure {{stakeholder.name}} will be notified of your response
- Note the timestamp of your reply for future reference

## HSM.03.5 Update your records

- If you created a WBS goal, use the agent-manager MCP tool to:
  - Access your WBS document (ID: {{documents.wbs.id}})
  - Add a note that you've replied to the stakeholder
  - Record the reply timestamp in the relevant goal
  - Note any commitments or deadlines you mentioned
  - Save the updated WBS document

## HSM.03.6 Monitor for response

- Keep the thread active in your awareness
- Be prepared to respond promptly if the stakeholder has follow-up questions
- If you're waiting for stakeholder input, use the agent-manager MCP tool to set the related WBS goal to BLOCKED
- Plan to check back on the thread periodically for any new messages through the agent-manager tool
